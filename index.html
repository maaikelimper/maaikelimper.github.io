WMO-IM GitHub

WMO-IM github is a working space for WMO experts to maintain and improve upon the WMO Information System (WIS).
Visit the WMO Community Site for official descriptions and to access published resources.
If you are a member of an Expert or Task Team, visit the wmo-im wiki for guidance and workflows.
